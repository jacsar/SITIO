PÁGINA DONDE SE MUESTRAN LOS NUMEROS



DESCRIPCION DEL PROCESO

A continuacion veremos como se muestran un listado de 10 valores numericos, los cuales han sido mostrados de forma automatica a travez de un bucle usando el lenguaje de JavaScript, como lenguaje de programación.

Se embebe el codigo JavaScript dentro del formato HTML, utilizando la etiqueta Script, se le indica a HTML que se usara una fracción de código JS, proximo, iniciamos con escribir el ciclo For, no declaro variables ya que automaticamente la estructura del ciclo nos da una variable incluida, la cual uso para este propodito, le decimos al ciclo que inicie en uno y que se detenga hasta que el valor de la variable sea menor o igual a 10, podria ser solo con el signo igual, y funcionaria de la misma manera, y que el incremento en cada paso sea de uno en uno.

luego le pedimos que imprima en pantalla el mensaje "ESTE ES EL NUMERO:" y seguido del numero concatenamos la variable del ciclo, para muestre las iteraciones del mismo, tambien le indicamos que imprima en pantalla un salto de linea, para evitar que nos queden los valores en linea, siendo asi, que nos quedan en bloque.
